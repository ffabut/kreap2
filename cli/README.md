# Command Line Interface

Work in progress... ale muze fungovat jako rozcestnik pro samostudium...

## Frameworky na prikazy
- https://github.com/pallets/click/ - framework pro tvorbu CLI
- https://github.com/fastapi/typer - framework pro tvorbu CLI, zajimave ramecky
- https://github.com/google/python-fire - automaticke generovani CLI z jakehokoliv Python objektu
- https://github.com/textualize/rich-cli - framework pro tvorbu CLI, od tvurcu a z ekosystemu Rich/Textual

## Barvy a formatovani textu

- https://github.com/Textualize/rich - barevny a formatovany text, od tvurcu rich-cli/textual

## Progress Bar & other animations

- https://github.com/rsalmei/alive-progress - progress bar a spinnery
- https://github.com/tqdm/tqdm - progress bar

- https://github.com/peterbrittain/asciimatics
